# 프로젝트 실행 방법

## 1. Strapi 서버 실행

1. 터미널을 열고 `backend` 디렉토리로 이동합니다.
   ```bash
   cd server
   ```
2. 의존성을 설치합니다.
   ```bash
   npm install
   ```
3. Strapi 개발 서버를 실행합니다.
   ```bash
   npm run develop
   ```

## 2. 프론트엔드 개발 서버 실행

1. 의존성을 설치합니다.
   ```bash
   npm install
   ```
2. 프론트엔드 개발 서버를 실행합니다.
   ```bash
   npm run dev
   ```

---

각 서버는 별도의 터미널에서 각각 실행해야 합니다.
